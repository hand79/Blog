電信業即時計費系統Domain knowledge

某電信業即時計費系統簡易系統設計Domain knowledge描述，大概整理下。

門號申裝流程

  1. 電信申租門號(月租型、預付卡)
  2. 選號(手機號碼)
  3. 服務方案、優惠(後頭供裝在特定門號優惠服務)
  4. 門號啟用(使門號生效進行打電話、上網等服務)

服務內容包括:

  • 資費設定、變更資費
  • 用量查詢
  • 計費周期(BillCycle)，出帳的週期
  • 加值服務(數據加包價購，預付卡儲值等)
  • 用量簡訊(到達特定亮時給予提醒)
  • 到期簡訊(服務到期、數據包到期、預付卡到期等)
  • 用罄簡訊(eg. 上網數據包用罄發送通知)
  • 服務群組設定
  • 優惠內容設定
